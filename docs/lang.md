# ブロックなどの名前を指定する

## 試しに、「mumupoppy:vhs」という名前のブロックを日本語で「VHS」と表示してみる

(リソースパック)/texts/ja_JP.lang という名前のフォルダを作る。

中に、<種類>.<名前空間>:<名前>=<つける名前>の形で、表示名を記載する。

```ja_JP.lang

tile.mumupoppy:vhs.name=VHS

```

## いろんな種類のものに名前をつける

種類は、アイテムはitem、エンティティはentityなど基本的にそのままだが、blockのみtileという名前になる。

エンティティの名前は、kill時のログなどに表示される。
